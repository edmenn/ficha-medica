Arquitectura — Opción A

Next.js 14 + Supabase + Vercel · 100% free tier

Capas del sistema

── CLIENTE (navegador / celular) ──────────────────────
Next.js 14 App Router   ← React 18, TypeScript, Tailwind CSS
PWA installable · responsive · captura de cámara nativa
── SERVIDOR (Vercel serverless) ───────────────────────
Next.js API Routes   ← /api/analyze · /api/records · /api/export
↓ llama a IA con la API key del usuario
Claude Vision API  |  Gemini Vision API
── BASE DE DATOS (Supabase) ────────────────────────────
PostgreSQL    ← registros, usuarios, campos personalizados
File Storage  ← imágenes originales (JPG/PNG/HEIC)
Auth          ← email/password + invitaciones
Row Level Security  ← cada usuario solo ve sus datos
