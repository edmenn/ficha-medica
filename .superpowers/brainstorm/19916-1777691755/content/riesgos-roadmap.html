Riesgos técnicos

ALTO
Calidad de extracción IA variable
Fotos borrosas, documentos poco estándar o escritura a mano reducen precisión.
✓ Mitigación: mostrar confianza por campo, destacar campos dudosos, nunca auto-guardar sin revisión.
ALTO
Timeout Vercel en análisis de imagen
Free tier: 10s máximo por API Route. OpenRouter puede tardar 5-15s con imágenes.
✓ Mitigación: comprimir imagen antes de enviar (<1MB), streaming de respuesta, o Supabase Edge Function sin límite de tiempo.
MEDIO
Seguridad API keys de usuarios
Keys de OpenRouter guardadas en DB, aunque encriptadas. Breach = exposición.
✓ Mitigación: AES-256 + clave en Vercel env, RLS en Supabase, key nunca en cliente.
MEDIO
Supabase free tier: límite storage
1GB storage gratis. ~500 fotos de 2MB = límite alcanzado rápido.
✓ Mitigación: comprimir imágenes a <500KB en cliente antes de subir (canvas API).
BAJO
HEIC desde iPhone no soportado en browser
iPhone guarda fotos en HEIC, no todos los navegadores lo manejan.
✓ Mitigación: convertir HEIC → JPEG en cliente con librería heic2any antes de subir.
